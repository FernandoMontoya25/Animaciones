Animaciones CSS

Transform

Traslate

Codigo de ejemplo

.traslate:hover {

   transform: translateX(120px);

}

Desplaza un elemento en el eje X y/o Y.

Scale

Codigo de ejemplo

.scale:active {

   transform: scaleX(3) scaleY(3);

}

Cambia el tamaño de un elemento en los ejes X y/o Y.

Rotate

Codigo de ejemplo

.rotate:active {

   transform: rotate(360deg);

}

Rota un elemento alrededor de un punto de origen.

Skew

Codigo de ejemplo

.skew:active {

   transform: skewX(30deg)    skewY(90deg);

}

Inclina un elemento en el eje X y/o Y.

Matrix

Es una forma más avanzada de aplicar transformaciones usando una matriz 2x3. Combina todas las transformaciones anteriores en una sola.

Perspective

Codigo de ejemplo

 .perspective:active {

   transform: perspective(500px)    rotateX(180deg);

}

Se utiliza principalmente con transformaciones 3D. Define la perspectiva desde la cual se ve un elemento 3D.

Transition

Transition-property

Codigo de ejemplo

.transitionProperty {

   transition-property: width;

}

.transitionProperty:active {

   width: 120px;

}

Especifica la propiedad que se va a moficiar por ejemplo..
color
width
height
etc

Transition-duration

Codigo de ejemplo

.transitionDuration {

   transition-property: width;
   transition-duration: 3s;

}

.transitionDuration:active {

   width: 200px;

}

Indica cuanto tiempo durara la animacion

Transition-timing-function

Codigo de ejemplo

.transitionTimimng {

   transition-property: width;
   transition-duration: 3s;
   transition-timing-function: ease;

}

.transitionTimimng:active {

   width: 200px;

}

Establece como se animara la propiedad a lo largo de la duracion ejemplo.

  • linear: Velocidad constante.
  • ease: Comienza lentamente, luego rapidamente y finaliza lentamente.
  • ease-in: Comienza lentamente.
  • ease-out: Finaliza lentamente.
  • ease-in-out: Comienza y finaliza lentamente.

Transition-delay

Codigo de ejemplo

.transitionDelay {

   transition-property: width;
   transition-duration: 3s;
   transition-timing-function: ease;
   transition-delay: 2s;

}

.transitionDelay:active {

   width: 200px;

}

Establece un tiempo de espera antes de que comience la animacion.

Transition-All

Codigo de ejemplo

.transitionAll {

   transition: width 3s ease 2s;

}

.transitionAll:active {

   width: 200px;

}
